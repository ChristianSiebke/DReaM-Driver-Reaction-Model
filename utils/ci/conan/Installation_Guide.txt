1. Install MSYS2 64-bit to C:/msys64 (default)
	https://www.msys2.org/
	
2. Open MSYS2 Shell and execute the following commands. Reply with "yes" or "return" if necessary.
	- $ yes | pacman -Syu (repeat until no additional packages where installed)
	- $ pacman -S --needed base-devel mingw-w64-x86_64-toolchain 
	- $ yes | pacman -S mingw-w64-x86_64-cmake mingw-w64-x86_64-boost mingw-w64-x86_64-protobuf mingw-w64-x86_64-gtest mingw-w64-x86_64-ccache mingw-w64-x86_64-doxygen mingw-w64-x86_64-graphviz mingw-w64-x86_64-qt5 git
	- $ pacman -S mingw-w64-x86_64-python-pip	
	- close MSYS2 shell
3.	- Add the following to you enviroment variables (Use "/" NOT "\")
		- CC=C:/msys64/mingw64/bin/gcc
		- CXX=C:/msys64/mingw64/bin/g++
	
4. Open MSYS2 MINGW64 Shell and execute the following commands. Reply with "yes" or "return" if necessary.
	- $ pip install pyodbc
	- $ pip install conan
	- $ conan profile new default --detect
	- $ conan profile update settings.compiler.libcxx=libstdc++11 default
	- navigate to "/ci/conan" in repo with cd
	- $ conan export ./recipe/osi openpass/testing
	- $ conan export ./recipe/fmi openpass/testing
	- $ conan export ./recipe/openpass
	- $ mkdir build && cd build
	- $ conan install .. --build=missing
	
Conan tips:
	- Install and build if something is binaries are missing
		$ conan install <path to conanfile.txt> --build=missing
	- CleanUp everything
		$ conan remove "*" -s -b -f
	- create new package
		$ conan new myPackage
	- build and export recipe to local cache
		$ conan create <path to conanfile.py> myPackage/<version>
	- export recipe to local cache
		$ conan export <path to conanfile.py> myPackage/<version>
	- show what packages are currently in the local cache
		$ conan search
	- show whatÂ´s in your current default profile
		$ conan profile show default
	- show dependencies 
		$ conan info <path to conanfile.txt>
